# 🛒 E-commerce Sales Analysis (SQL)

## 🔹 Overview
This project analyzes sales data using SQL.

## 🔹 Tools Used
- MySQL

## 🔹 Data Cleaning
- Removed null values
- Fixed date format
- Removed invalid data

## 🔹 Analysis
- Total Sales & Profit
- Top Products
- Region-wise analysis
- Monthly trends

## 🔹 Insights
- Some products generate high revenue
- Certain regions perform better
